WPCargo Receiving allows you to update multiple shipments’ status through barcode scanner device or using the bulk update button provided in Frontend Manager.

Any Bluetooth or wired barcode reader devices can be used in these features as long as they are compatible with the desktop you want to connect.

How it works:

Simply assign users and set the shipping history fields before you scanned the barcode with your device. Once read you hear a beep and it will automatically save the fields you assigned or set into the corresponding tracking number.
